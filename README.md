Gokatas is a command line tool to show the Go katas.

```
$ go install github.com/gokatas/gokatas@latest

$ gokatas 
Name     Description               Topics                         URL
----     -----------               ------                         ---
areader  io.Reader implementation  interfaces                     git@github.com:gokatas/areader.git
boring   concurrency patterns      channels, goroutines, select   git@github.com:gokatas/boring.git
shift    simple cipher             cryptography, design, testing  git@github.com:gokatas/shift.git
```
