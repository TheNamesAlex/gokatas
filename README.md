Gokatas is a command line tool to show the Go katas.

```
$ go install github.com/gokatas/gokatas@latest

$ gokatas 
Name     Size  Stars  Description                      Topics                         URL
----     ----  -----  -----------                      ------                         ---
areader  3     0      io.Reader implementation         interfaces                     git@github.com:gokatas/areader.git
battery  1     0      parse external command's output  exec, regexp                   git@github.com:gokatas/battery.git
boring   6     1      concurrency patterns             channels, goroutines, select   git@github.com:gokatas/boring.git
shift    3     0      simple cipher                    cryptography, design, testing  git@github.com:gokatas/shift.git
```
